Lesson 1: The Basics of the Web and HTML

The Basics of the Web

World Wide Web

The World Wide Web is a collection of HTML documents. HTML is the basis for almost every webpage and the links between the pages called hyperlinks.

Major pieces of the web:

  • HTTP: The main protocol of the web
  • Servers: Computers that host the files that make up the web
  • Internet: The world's largest computer network
  • Browser: A program that runs on your computer to display files found on the web
HTML
HTML stands for HyperText Markup Language.
HTML is made up of:
  • text content
  • markup
  • references to other documents
  • links to other pages

HTML Markup

HTML markup is made up of things called tags. An HTML tag is always contained within angled brackets < >. Most tags have an opening tag and closing tag .Some tags do not require a closing tag called “void” tag.

* b- Tag : Anything that appers inside the b-tag appears bold .
* em- Tag : This tag makes things italics .

HTML Attributes

Attribute is a property of an HTML element. Element refers to everything within a set of opening and closing tags. HTML tags can have attributes that determine the behaviour of the tag.

An example of a tag that uses attributes is the anchor tag : “href”. Another tag that uses attributes is the image tag: “src”

All spaces in text, whitespaces, are largely ignored in HTML.To force out HTML to have multiple lines we can use break tag and paragraph tag.

Inline vs. block tags

HTML elements are either inline or block.

  • Inline : Shows texts inline.
  • Block : Elements are in an invisible box.

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repetition

Adding CSS style to HTML structure is a great way to avoid repetition so programmers do not have to write the same code over and over again. It also avoids copy and pasting certain style attributes into many HTML elements.

For example: If I want to change the color of all h3 elements in my web page, then instead of manually changing the h3 elements color, adding CSS style to HTML structure makes it easy to change style of my web page So, it gives consistency to my web page and avoiding errors.

Understanding CSS

CSS stands for Cascading Style Sheets and they give programmers a way to control the style of related HTML elements. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.

Inheritance: Inheritance is a key feature in CSS. It relies on the ancestor- descendant relationship to operate.

Given the following style sheet:

h5 {color : Peru;} -- suppose there is an h5 element with an emphasizing element ⟨em⟩ inside: the emphasized word will inherit the color of the parent element, h5.

This to illustrate inheritance.

What is a variable in Python?

A Variable is a named memory location that can hold a value. Variable name can be any sequence of letters and numbers, as well as underscores. Variable must start with a letter or an underscore. Variable cannot start with a number, cannot have space in it and cannot have math symbols in it.

What are some ways variables are useful?

Variables can be useful to programmers in many ways:

  • Variables play an important role in computer programming because they enable programmers to write flexible programs. Rather than entering data directly into a program, a programmer can use variables to represent the data. Then, when the program is executed, the variables are replaced with real data. This makes it possible for the same program to processdifferent sets of data.
  • They give us a way to store the value of important data. A variable's data type indicates what sort of value the variable represents, such as whether it is an integer, a floating-point number, or a character.
  • They give us a way to change the value of something. When you assign a value to a variable, you replace the old value with a new one. The old value is gone forever. You can, and often do, replace the old value of a variable based on the current value of the variable.
What is the difference between 5 + 6 and "5" + "6"?

In Python, anything in "quotes" called a string. A string is just a sequence of characters surronded by quotes. We can use single quote (' ') and double quote (" "), but cannot mix quotes. If you start with single quote,it has to end with single quote, and if you start with double quote, it has to end with double quote.

So, when we do 5 + 6 --- output will be 11.
